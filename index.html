K-8
Key Decisions — Social Studies Done
  • 1. Standards: Adopt TEKS as baseline K-8 SS standards across all Alpha campuses. 92.5% national overlap confirmed across 6 states. Short-term: TEKS backbone + state-history modules at G4 and G7. Long-term: "Universal-Plus" — TEKS + non-state-specific content from CA, FL, MA, NY, TN
  • 2. Testing: No external national SS test works for a multi-state charter network — Alpha builds its own assessment framework. STAAR G8 blueprint adapted downward for G3-5. TEKS throughout (60-70% Readiness, 30-40% Supporting). Externally sourced questions only. G8 summative = STAAR
  • 3. App Stack (Short-Term): G3-5: BrainPOP instruction + PP100 mastery loop (Study Island + generated IXL-style questions). G6-8: Edmentum Courseware — credit-bearing, TEKS-aligned. This is patchwork — fills the gap but is not a long-term curriculum
  • 4. Long-Term Curriculum: Build Alpha's own K-8 SS curriculum — everything on the market is teacher-dependent or incomplete. CKHG as content base + BrainPOP/YouTube for video + BRI for civics. Architecture: TEKS-Structured Modular (recommended). Multi-month build, on hold pending engineering
  • 5. Learning Outcomes: SS contributes to Alpha's 2x acceleration goal. Performance measured via PP100 mastery (G3-5), 80% accuracy on Mastery/Unit Tests (G6-8 Edmentum), TEKS-aligned end-of-grade assessments (G3-8), STAAR G8 as external summative benchmark
  • 6. XP: 1 XP = 1 min active learning. G3 BrainPOP ~1,020 XP (vs old IXL-only ~738 XP). G6-8 Edmentum 4,275 XP. BRI Civics 450 XP
  • 7. Assessment System: Replace manual one-off test creation with centralized, automated, framework-enforced assessment generation. Framework-agnostic, standards-agnostic, content-source-agnostic. G3 TEKS MVP scoped. On hold pending engineering
